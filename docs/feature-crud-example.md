# CRUD Example

Buatkan Fitur CRUD untuk menambahkan, menghapus, mengedit, dan melihat data TODO. Terapkan fitur tersebut pada halaman TODO. Terdapat fitur pencarian, filter, dan pagination data TODO.

## Langkah-langkah

1. Buatkan schema untuk tabel TODO. Data TODO terdiri dari id, title, description, status, dan created_at.
2. Buatkan migration untuk tabel TODO.
3. Implementasikan pada halaman TODO (pages/todo/index.vue).
4. Gunakan Nuxt UI Component untuk membuat antarmuka yang lebih menarik. Referensi https://ui.nuxt.com/docs/components.
5. Implementasikan fitur pencarian, filter, dan pagination data TODO.
6. Gunakan Tabel dari Nuxt UI Component untuk menampilkan data TODO.
7. Implementasikan fitur tambah, edit, dan hapus data TODO dengan menggunakan Modal dari Nuxt UI Component. Tambahkan juga fitur bulk create dan bulk delete.
8. Gunakan `vee-validate` dan `zod` untuk validasi form.
9. Gunakan API untuk mengakses data TODO.
